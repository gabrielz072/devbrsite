# Site de Artigos de Tecnologia

Este projeto é um site simples de artigos sobre tecnologia, desenvolvido como parte do meu aprendizado na área de desenvolvimento web.

Sou estudante e criei este site com o objetivo de praticar HTML, CSS e conceitos básicos de estruturação de conteúdo, responsividade e boas práticas para a web.

## Objetivo do Projeto

- Praticar desenvolvimento front-end
- Organizar artigos de tecnologia de forma clara e acessível
- Aprender boas práticas de HTML semântico e CSS
- Criar um projeto simples, mas evolutivo

## Tecnologias Utilizadas

- HTML5
- CSS3

## Conteúdo

O site é focado em artigos sobre tecnologia, abordando temas como:

- Desenvolvimento web
- Conceitos básicos de programação
- Explicações técnicas em linguagem acessível
